# Security Policy

## Reporting a vulnerability

Issues are disabled on this repository. To report a security vulnerability, please use [GitHub's private vulnerability reporting](https://github.com/adeaap/render-modes/security/advisories/new) instead of opening a public issue or PR.

You can expect:

- An acknowledgement within a reasonable timeframe.
- A fix or mitigation if the report is confirmed and applies to this project.

This is a personal educational project, so response times may vary. Please do not publicly disclose a vulnerability until it has been addressed.
